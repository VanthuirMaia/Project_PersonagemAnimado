## Sobre o Projeto

Este projeto foi desenvolvido como trabalho final da disciplina **IA Generativa para Mídia Visual**, parte do programa de **Residência em IA Generativa** da **Universidade de Pernambuco (UPE)**.

O objetivo é implementar um pipeline completo de IA Generativa para:
1. Criar personagens visuais consistentes usando Stable Diffusion
2. Gerar animação curta em vídeo preservando identidade visual
3. Documentar tecnicamente todo o processo

## Tecnologias Principais

- Stable Diffusion (Hugging Face Diffusers)
- PyTorch
- OpenCV
- Streamlit
- Python 3.8+
